# siddhartha-river-of-life-test
A mobile-first interactive web experience inspired by Hermann Hesse's Siddhartha. Participants answer 8 questions to discover their current "river state" of life, receive personalized interpretations, and engage in reflective discussion during reading club events.一个受《悉达多》启发的移动端互动网页。参与者通过8道题发现自己当前所处的生命河流阶段，获得个性化解读，并用于读书会讨论和交流。
